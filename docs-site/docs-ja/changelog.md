# 変更履歴

## v0.3.2

- Linux 対応追加（amd64、arm64）
- プロキシ対応追加：SOCKS5/SOCKS4/HTTP/HTTPS プロキシトンネル
- SFTP ブレークポイントレジューム追加（`-resume` フラグ）
- hash および verify サブコマンド追加（md5、sha1、sha256、sha512）
- WinGet インストール対応追加
- プロキシタイムアウト処理の改善
- 初回ドキュメントサイト

## v0.3.1

- SFTP 転送中のタイムアウトリセットの修正
- rz/sz ファイル転送の安定性向上

## v0.3.0

- SDK パッケージと CLI エントリーポイントの抽出
- `WithProgress`、`WithFileSelector`、`WithSignalHandler` オプションの追加
- SDK は UI コードを含まなくなりました。CLI 側のアダプターは独立して実装

## v0.2.0

- SCP スタイル転送サポートの追加
- Rsync スタイル転送サポートの追加
- 設定ファイルサポートの追加
- 指数バックオフ付き接続リトライの追加

## v0.1.0

- 初回リリース
- SSH パスワード/キー認証
- インタラクティブシェル（ローミングターミナルモード）
- SFTP ファイル転送
- 動的ターミナルリサイズ
- rz/sz シェル転送
